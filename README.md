# Programming Languages Timeline / Línea del Tiempo de los Lenguajes de Programación

An elegant, interactive, and educational timeline of programming languages, their history, achievements, and impact.
Una línea del tiempo elegante, interactiva y didáctica de los lenguajes de programación, su historia, logros e impacto.

## Project Structure / Estructura del Proyecto

This project is a frontend-only web application designed to be deployed on GitHub Pages.
Este proyecto es una aplicación web puramente frontend diseñada para ser desplegada en GitHub Pages.
